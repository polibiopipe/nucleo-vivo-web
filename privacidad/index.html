Privacidad

Esta página explica de forma clara qué datos puede recibir Núcleo Vivo a través de su sitio institucional y enlaces de contacto.

Datos personales

Recibimos solo la información que decides enviarnos.

Este sitio no incluye formularios activos, pagos ni registro público de usuarios. Cuando haces clic en un correo, tu mensaje se gestiona desde tu propio cliente de email.

Si escribes a Núcleo Vivo, podremos recibir tu nombre, correo, organización, asunto y el contenido que decidas compartir para responder tu consulta.

El acceso a servicios externos, como Spotify o Escucha Viva, se rige también por las políticas de esas plataformas.

Uso de información

Uso responsable y acotado.

La información recibida se utiliza para responder consultas, coordinar conversaciones, revisar propuestas de colaboración o gestionar solicitudes formativas.

  1. 01Contacto directo

    Usamos tu mensaje para responder la solicitud que enviaste.

  2. 02Sin venta de datos

    No vendemos bases de datos ni compartimos información con terceros para publicidad.

  3. 03Actualización

    Puedes escribir para solicitar corrección o eliminación de datos enviados por correo.
