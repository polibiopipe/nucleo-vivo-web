Mi Aula

Un punto de acceso para organizar rutas formativas, materiales y seguimiento educativo de Sembrar cuando una experiencia o cohorte esté activa.

Acceso responsable No hay un formulario de registro público ni login simulado en esta versión.
  • Acceso por rutas o cohortes definidas
  • Materiales y seguimiento formativo
  • Contacto directo para activaciones
Qué reúne

Un espacio para ordenar el aprendizaje sin prometer una plataforma que todavía no está conectada.

Mi Aula queda preparada como ruta pública para participantes de experiencias formativas. Su función es concentrar el acceso a cursos, materiales, orientaciones y seguimiento cuando Núcleo Vivo active una ruta con una organización, comunidad o grupo específico.

Esta página evita crear registros o evaluaciones falsas. Cuando exista una plataforma con autenticación, progreso, actividades y resultados reales, el acceso quedará conectado desde aquí.

Uso previsto

Cómo se integrará a las rutas de Sembrar.

El acceso formativo se activa cuando hay contenidos, acompañamiento y condiciones claras para participar.

  1. 01Ruta o curso definido

    La persona conoce objetivos, duración, materiales y modalidad.

  2. 02Acceso habilitado

    La participación se activa por invitación o coordinación directa.

  3. 03Seguimiento formativo

    El avance se acompaña con recursos, actividades y orientaciones reales.

¿Quieres activar una ruta formativa?

Podemos conversar sobre cursos, talleres o experiencias educativas para tu comunidad u organización.
